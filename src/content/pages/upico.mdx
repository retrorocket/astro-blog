Twitter のアイコンを API 経由でアップロードするツール（upico）は Twitter API が使用できなくなったため 2023/04/30 をもって公開を終了しました。
長い間ご利用ありがとうございました。

## …と思ったんだけど

- なんかまだ API v1.1 が動くらしいのでがんばって復旧しました。使いたい方は自己責任でどうぞ。
- サーバー維持が大変になってきたので[カンパ](/kampa)していただけると嬉しいです。
  - あまりにも維持がきついため、このアプリに関しては広告の数をむちゃくちゃに増やしています。ご了承ください。（それでもペイできていません。）
- **このツールがいつまで使用できるかについては、Twitterからアナウンスがないため開発者にもわかりません。**
