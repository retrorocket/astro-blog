## 🐧 Twitter 関連

#### [API 経由でアイコン画像をアップロードするツール](/upico)

Twitter のアイコン画像をアイコン画像を API 経由でアップロードすることで、アイコン画像の JPEG 化の回避や透過情報の保持を行うツール。アップロードした画像を PNG32 に変換する機能もついています。
2023/04/30 にサービスを終了したのですが、API v1.1 がまだ動くので暫定的に復旧させました。

月のリクエスト数は 1 万 5000 くらいのようです。

## 🐤 Twitter API が使用不可になったため公開終了

自分が欲しい機能を実現したり、Twitter の制限を回避するためのツールが多かったです。

#### [favico! – Twitter のアイコンをふぁぼれるサービス](https://favico.retrorocket.biz/)

Twitter のアイコンをお気に入りに登録できるサービス。自分が使いたくて作ったサービスで思い入れが半端無かったため、[自分が登録したお気に入り情報](https://favico.retrorocket.biz/retrorocket)だけ残してあります。ブラウザからワンクリックでお気に入り登録するための Chrome 拡張もありました。

<Image src={favico_screen} alt="favico! のスクリーンショット" />

#### あなたが一番ふぁぼってる人

Twitter での直近 200 件のふぁぼを取得して、自分が一番ふぁぼってる人を表示するツール。ほんの一瞬だけバズったりしたこともありました。

#### Twitter に画像を PNG でアップロードするツール

画像を左上 1px を 1%分だけ透明にし、Twitter に PNG32 でアップロードするためのツール。普通にアップロードすると JPEG 化されるため、それを回避するために作成しました。

#### TimeLine Copier – Twitter でフォローしてる人をリストにコピーするツール

Twitter でフォローしてる人やフォロワーをリストにコピーするツール。サードパーティ製の Twitter クライアントではタイムラインの取得に回数制限があるため、リスト化してその制限を回避するために作成しました。

#### [はっしゅたんぐら。 – twitter # to tumblr.](https://tag.retrorocket.biz/)

20 分おきくらいに、指定された#ハッシュタグがつけられたツイートと投稿時刻を取得し、自動で tumblr.に投稿するツール。ハッシュタグの内容を日記に投稿したいという人の[ツイートを見て](https://retrorocket.biz/archives/198)作成しました。

## 😢 公開終了

#### @Twitter Profile summary Canceller

Twitter で他の人のプロフィールをクリックしたときに表示されるモーダルウィンドウを無効化するための拡張。Twitter の仕様変更で不要になったため公開終了。

#### 君のつぶやきだけ見せて

Twitter で他の人のプロフィールページに表示されているツイートから、メンションと RT を除外するための拡張。Twitter の仕様変更で不要になったため公開終了。

#### mio chang Coupon Switcher

ブラウザから IIJmio の API を使えるようにするサイト。クーポン切り替えアプリが使用できない環境だったので作成しました。
お世話になっていた「IIJmioクーポンスイッチAPI」が [2025年1月31日をもって提供終了](https://www.iijmio.jp/info/iij/1719529607.html)となったため、こちらも終了しました。
認証認可の仕組みがOAuth 2.0 Authorization Framework (Implicit Grant) だったため、クライアントを作成するにあたってとても勉強になりました。ありがとうございました。
